# Fatoumata Soumare's Portfolio

This project is a personal portfolio website for Fatoumata Soumare, showcasing her skills and experiences in a visually engaging manner. The website features a unique loading screen with twinkling stars, a beautifully animated name display, and an interactive navigation system using planets.

## Features

- **Loading Screen**: A black loading screen with twinkling stars that enhances the user experience during the initial loading phase.
- **Animated Name**: The name "Fatoumata Soumare" is displayed in a paintbrush style, with an animation that draws the name on the screen.
- **Rotating Image**: A rounded square profile image that rotates after the name fades out, adding a dynamic visual element.
- **Planet Navigation**: A navigation bar replaced with clickable planets that link to different sections of the portfolio, providing a fun and thematic navigation experience.
- **Typewriter Effect**: The text "marketer..." is displayed below the rotating image with a typewriter animation, emphasizing Fatoumata's professional identity.
- **Welcome Message**: A warm welcome message is displayed at the top left of the page, inviting visitors to explore the portfolio.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/fatoumata-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd fatoumata-portfolio
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage

Open your browser and go to `http://localhost:3000` to view the portfolio. The loading screen will appear first, followed by the animated name and profile image. Use the planets in the navigation to explore different sections of the portfolio.
